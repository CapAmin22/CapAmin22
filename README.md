# Hi, I'm Amin 👋

I build AI-powered products and ship evaluation-driven systems that marketing and sales teams can actually use.

Most of my work is hands-on: real product thinking, agentic architectures, RAG pipelines, structured evals, and human-in-the-loop workflows applied to AI Marketing SaaS, Fin-Tech, and B2B intelligence.

I run the open-source [Creative Intelligence Portfolio](https://github.com/CapAmin22?tab=repositories) — a 6-project system where each repo solves one layer of the creative-performance stack, from ad classification to attribution, composed into a production-shaped orchestration system.

### What I work on

- **AI Product Management**: problem framing, eval design, workflow orchestration, agentic fundamentals
- **Creative Intelligence**: ad classification → brand RAG → creative fingerprinting → network alignment → attribution loops
- **B2B Sales Intelligence**: prospect enrichment pipelines, agentic lead extraction, trust-centric outreach
- **Evaluation-First Building**: golden sets, LLM-as-judge, RAGAS, calibration curves — every claim ships with evidence
- **Agentic Systems**: LangGraph state machines, multi-agent orchestration, human-in-the-loop interrupts, recovery states

## About Me

- **AI Product Manager** building at the intersection of LLMs, marketing intelligence, and product rigor — focused on systems that learn from outcomes, not just generate content.

- **Builder-PM philosophy**: I don't just write PRDs — I ship code, build eval harnesses, and prove quality with numbers. Every project in my portfolio has a structured evaluation layer and a failure-mode teardown.

- **6-Project Creative Intelligence System**: Designed and built an end-to-end creative-performance stack — [HookLens](https://github.com/CapAmin22/HOOKLENS) (classifier) → [BrandGround](https://github.com/CapAmin22/BRANDGROUND-) (RAG) → [CreativeOS](https://github.com/CapAmin22/CreativeOS) (orchestrator) → [CreativeSignal](https://github.com/CapAmin22/CreativeSignal) (fingerprinting) → [SignalForge](https://github.com/CapAmin22/SignalForge-) (network alignment) → [AttributionOS](https://github.com/CapAmin22/AttributionOS-) (measurement loop).

- **B2B Intelligence**: Built [TrueSignall](https://github.com/CapAmin22/TrueSignall), a zero-cost prospect intelligence engine, and [LinkedIn Post Leads](https://github.com/CapAmin22/Linkedin_Post_Leads), an agentic lead extraction system.

🚀 Building what's next in AI-powered marketing intelligence and product-led growth. Always open to collaborate on eval-driven AI products.

## 🏗️ Flagship Projects

### Creative Intelligence Stack *(6-project system)*

| Module | Repo | What it does | Stack |
|--------|------|-------------|-------|
| 🔍 **Classify** | [HookLens](https://github.com/CapAmin22/HOOKLENS) | Paste a marketing hook → typed analysis: archetype, funnel stage, emotional angle, brand-voice score. Wrapped in an eval harness with human-in-the-loop trust thresholds. | Python · Streamlit · LLM-as-Judge |
| 📚 **Ground** | [BrandGround](https://github.com/CapAmin22/BRANDGROUND-) | RAG over brand guidelines + competitor ads → cited answers with retrieval & faithfulness evals + a written failure-mode teardown. | LangChain · ChromaDB · RAGAS |
| 🧠 **Orchestrate** | [CreativeOS](https://github.com/CapAmin22/CreativeOS) | End-to-end creative intelligence: ingest → cluster → score → generate variants → eval-gate → human review → feedback loop. LangGraph state machine. | LangGraph · FastAPI · Docker |
| 🎬 **Perceive** | [CreativeSignal](https://github.com/CapAmin22/CreativeSignal) | Raw ad creative → machine-readable fingerprint: scenes, hook timing, transcript, tone, emotional arc, visual style. | ffmpeg · Whisper · Vision LLM |
| 📡 **Distribute** | [SignalForge](https://github.com/CapAmin22/SignalForge-) | Map creative fingerprints to ad-network delivery behaviors. Pre-flight reads with hypothesis-flagged confidence. | Python · Embeddings |
| 📊 **Measure** | [AttributionOS](https://github.com/CapAmin22/AttributionOS-) | Creative-level attribution: link creative fingerprints to performance data, detect fatigue, feed back to the stack. The data flywheel. | Python · Analytics |
